Capabilities

Active OS
Intelligence.

ADITYA has direct, secure access to interact with your machine, automate workflows, and optimise hardware in real time.

OS Control

Manage volume, brightness, sleep, hibernate, or lock your PC — all via natural voice commands.

Physical Automation

Trigger clicks, cursor movement, execute custom hotkeys, or type text using detached automation scripts.

OCR Targeting

Read elements on your screen using Tesseract OCR, processing text with low-contrast filters for precise interaction.

Game Booster

Suspend search indexers, raise active task priority, and get live system telemetry tracking for peak performance.

Voice Engine

Zero-latency voice recognition with multi-speaker diarization and context-aware conversational AI responses.

Plugin System

Modular skill-based architecture. Extend ADITYA with custom Python plugins, scripts, and automation modules.

Design

Decoupled
Architecture.

ADITYA separates concerns into independent, hot-swappable modules that communicate through a unified event bus.

01

Electron Client

Premium glassmorphism UI built with HTML/CSS/JS, featuring real-time particle canvases and custom widget panels.

02

Python Backend

Flask-based API server handling AI inference, system calls, hardware monitoring, and voice-to-text pipelines.

03

Skills Engine

Modular plugin loader that discovers and activates Python automation scripts at runtime with hot-reloading.

Interface

Premium Visual
Dashboard.

Get Started

Install in
seconds.

One command. No dependencies. Built natively for Windows.

PowerShell
PS > irm "https://raw.githubusercontent.com/ThisisDarkNova/Aditya-Ai/main/install.ps1" | iex
Command Prompt
C:\> powershell -ExecutionPolicy Bypass -Command "irm 'https://raw.githubusercontent.com/ThisisDarkNova/Aditya-Ai/main/install.ps1' | iex"
Winget
> winget install Aditya.AI
Scoop
> scoop install aditya-ai
Chocolatey
C:\> choco install aditya-ai
npm
> npm install -g aditya-ai
Yarn
> yarn global add aditya-ai
pnpm
