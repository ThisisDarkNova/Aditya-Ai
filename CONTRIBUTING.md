# Contributing to ADITYA OS

Thank you for helping to improve ADITYA OS! Here's how you can contribute:

## 🐛 Reporting Bugs
- Use the **Bug Report** template to submit issues.
- Please attach the relevant logs from `%APPDATA%/ADITYA/logs/electron.log` or `launcher.log` to make debugging easier.

## 🚀 Proposing Enhancements
- Use the **Feature Request** template to submit suggestions.
- Explain the problem it solves, how it works, and how it aligns with the Jarvis-level persona of ADITYA.

## 🛠️ Code Contributions
1. Fork the repo and create your branch (`feature/your-feature-name`).
2. Make sure the backend compiles correctly with `pyinstaller` without errors.
3. Test locally in dev mode using `npm run electron:dev` inside `/client`.
4. Open a Pull Request using the template provided.
