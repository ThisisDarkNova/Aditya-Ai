# Support Guidelines

Thank you for using ADITYA OS! Here are the ways you can get help or support:

## 💬 Community Support
- **Discussions**: For general questions, configuration help, and sharing ideas, please use [GitHub Discussions](https://github.com/DarkNova/ADITYA/discussions).
- **FAQ**: Check the [README.md](README.md) for quick troubleshooting guidelines.

## 🐛 Bug Reports
If you encounter a bug, please check the existing issues or open a new one using the **Bug Report** template.

## 🔒 Security Issues
Please refer to our [Security Policy](SECURITY.md) to report vulnerabilities privately.
