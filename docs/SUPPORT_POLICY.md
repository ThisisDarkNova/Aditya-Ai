# Support Policy & Maintenance SLA

We offer structured maintenance intervals for **VESPERA OS** users and developers.

## Support Scope

### What We Support
- Setup and installation instructions detailed in the [README](../README.md) and [DEVELOPER_GUIDE](DEVELOPER_GUIDE.md).
- Bugs and crashes relating to core modules (app resolver, event bus, memory system).
- Orb WebGL compatibility lints.

### What We Do Not Support
- Unofficial hardware configurations or custom firmware.
- Third-party python module integrations not declared in `requirements.txt`.

## Response SLA

- **Critical Bugs (security/crashes)**: Restored or hotfixed within 48 business hours.
- **Feature Requests**: Reviewed and triaged during bi-weekly maintainer meetings.
- **General Questions**: Addressed asynchronously on GitHub Discussions.
