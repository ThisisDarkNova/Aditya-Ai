# Vespera: The Marginalia 👻⌨️

> Ultra-luxury local AI integration for VS Code, powered by umbracore.

Vespera is an elite coding companion designed to feel like an extension of your own central nervous system. Rather than providing generic autocomplete popups, it works silently in the background, utilizing context-aware Generative AI to assist you with elite-tier precision.

## 💎 Features

* **Code Telepathy (Marginalia)**: Press `Ctrl+Shift+A` (or `Cmd+Shift+A` on macOS) to instantly summon Vespera. Based on the surrounding lines of code, Vespera will think, predict, and inject the logical next lines of code directly at your cursor.
* **Contextual Analysis (Hover)**: Hover over any symbol, function, or programming concept to receive a premium, 1-sentence analytical breakdown of its purpose.
* **Local-First & Secure**: Built to prioritize privacy, keys and configurations are retrieved locally and securely from your system settings with zero cloud telemetry.

## 🛠️ Configuration

Configure your Gemini API Key in the settings:
1. Open Settings in VS Code (`Ctrl+,`).
2. Search for `Vespera`.
3. Enter your Gemini API Key under `Vespera: Gemini Api Key`.
