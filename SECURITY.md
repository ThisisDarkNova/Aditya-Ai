# Security Policy

## Supported Versions
Only the latest master branch is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability
Aditya is a **Local-First** AI ecosystem. If you discover a vulnerability that allows for Remote Code Execution (RCE), WebSocket escaping, or unauthorized local telemetry access, please report it immediately.

**Do not open a public issue.**
Instead, please email the details to the core maintainers. We take privacy extremely seriously—Aditya is built to protect the user's local hardware, and any breaches of the `PhantomEngine` WebSocket protocols will be addressed within 24 hours.
